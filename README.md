# Synapse AI Toolkit

**SynapseAI** is a powerful software solution that harnesses the capabilities of Artificial Intelligence (AI) to help users create and manage their own informational assets. The software is designed to provide a range of possibilities for users to gather and analyze data, while making them easily traceable.

With SynapseAI, users can take advantage of the OpenAI API to gain access to a wide variety of advanced AI features, such as natural language processing and deep learning. These features allow users to easily analyze vast amounts of data, extract insights, and generate reports that can be used to drive strategic decision-making.

In addition, SynapseAI provides users with a user-friendly interface that makes it easy to navigate and use the various features of the software. Whether you are a curious mind, a researcher, a business owner, or simply someone who needs to manage large amounts of data, SynapseAI can help you achieve your goals more efficiently and effectively.

## Pre-requisites
Before you can run this project, make sure you have the following installed:

* NodeJS (v18 or higher)
* Yarn (v3.5 or higher)
* Nodemon
* Environment variables

## Installation
To get started, clone this repository and install the dependencies:

```
git clone https://github.com/ethx42/synapseAI.git
cd synapseAI
yarn
```
### Usage
To start the development servers for both packages, run:
```
yarn start
```
This will start both the client and server development servers in watch mode, using `nodemon` and `react-scripts`.

### To build both packages, run:

```
yarn build
```
This will build both the client and server packages and output the build artifacts to the dist folder for each package.

### Linting
To run linting for both packages, run:

```
yarn lint
```
This will run linting for both the client and server packages.
